Nautilus
========

Is the __default__ file manager in in Ubuntu 14.04

You can instqall it:

    sudo apt-get install nautilus



Open terminal here
------------------

```
sudo add-apt-repository universe
sudo apt-get update
sudo apt-get install nautilus-open-terminal
sudo apt-get install nautilus-actions
```

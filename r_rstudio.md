RStudio
=======

Some dependencies
------------------

    sudo apt-get install libjpeg62


Install RStudio
---------------

Download from 
<http://www.rstudio.com/products/rstudio/download/>

And run: 

    sudo dpkg -i rstudio-*****-amd64.deb 

Preview versions are available at:
<https://www.rstudio.com/products/rstudio/download/preview/>

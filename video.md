DVD
=========================================================================================

https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs

    sudo apt-get install libdvdread4
    sudo /usr/share/doc/libdvdread4/install-css.sh
    sudo apt-get install vlc

Rebooting may be necessary. 


Video Formats
=========================================================================================
